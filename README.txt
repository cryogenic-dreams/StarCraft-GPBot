StarCraft:Brood War Genetic Programmed with Koza fitness Artificial Intelligence using a Java Implementation 
( or  SCBWGPKAIJI  ([->provisional name<-]))
Is a bot that is currently under development, it should play StarCraft at some point in time.

In order to make it work you'll need:

1. A copy of StarCraft 1 with its expansion: Brood War.
2. BWAPI library and Chaos launcher.
3. BWMirror library.
4. JDK 8 32 bits.
5. SSCAI maps pack.
6. ECJ library.
7. GPBot folder (MightBeUseful is not necessary).

I'll update with links, and more indications of how to install and make it work as soon as I can.
